Line chart

Created line chart in Tableau and developed it using d3.js

This page shows visualization of line chart. Used tableau to create it.
Kept x-axis as Activity period, which was further filtered out to get months in year 2019 and y-axis as passenger count.

Visualizations

Caption: The trend of maximum number of Passenger Count for Activity Period Month.
Data Source := https://data.sfgov.org/Transportation/Air-Traffic-Passenger-Statistics/rkru-6vcg/data

Writeups

Passenger count is encoded along y-axis, and x-axis has time period. line represent maximum number of passengers travlled over time period. Used d3 and Air-Traffic-Passenger-Statistics to wrangle dataset. Filter out dataset using the source link. Manipulated Activity_period field from "text" to date type and converted Passenger_Count from text to number using JavaScript. This visualization can conclude that in the month of July and August maximum number of passengers travelled because at this time, there are students flying internationally, as well as domestic people get back from their summer break.
